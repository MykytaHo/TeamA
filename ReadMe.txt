# Local Services Marketplace - Team A

## Quick Start

Run from the root directory:
```bash
npm install my-app
npm run dev
```

Or manually:
```bash
cd my-app
npm install
npm run dev
```

## Project Structure

- **my-app/** - React + Vite application
  - **src/pages/** - Empty page components
  - **src/components/** - Navigation & other components  
  - **src/App.jsx** - Main app with routing & Firebase auth

## Available Scripts (from root)

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

App runs at http://localhost:5173/
